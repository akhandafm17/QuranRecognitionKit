# Model Notice

QuranRecognitionKit includes a bundled ONNX model artifact for offline Quran verse recognition.

The bundled model is based on NVIDIA's `nvidia/stt_ar_fastconformer_hybrid_large_pcd_v1.0` Arabic FastConformer model. NVIDIA's model card lists the model license as CC-BY-4.0 and states that the model is ready for commercial and non-commercial use.

Upstream model:

```text
nvidia/stt_ar_fastconformer_hybrid_large_pcd_v1.0
https://huggingface.co/nvidia/stt_ar_fastconformer_hybrid_large_pcd_v1.0
```
